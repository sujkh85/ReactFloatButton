Start
----------
    git clone https://github.com/sujkh85/ReactFloatButton.git
    cd ReactFloatButton
    npm install
    npm start


Dependencies
----------
    jquery@1.12.3 or jquery@2.2.3
    react@0.14.8

Use
-----------
    if you use react-float-button in project
    please use src/floatingButton.js and src/main.js
